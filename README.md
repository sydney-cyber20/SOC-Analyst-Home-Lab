# SOC Analyst Home Lab

This repository contains hands-on cybersecurity projects focused on SOC analyst skills, endpoint monitoring, vulnerability assessment, log analysis, and security investigation practice.

## Projects

### Project 01: Vulnerability Assessment Lab

A vulnerability assessment project focused on identifying and documenting security weaknesses in a lab environment.

### Project 02: Windows Sysmon Endpoint Monitoring Lab

A Windows endpoint monitoring project using Sysmon and Event Viewer to collect endpoint logs, generate system activity, and analyze Sysmon events.

## Skills Demonstrated

* Windows endpoint monitoring
* Sysmon installation and configuration
* Event Viewer log analysis
* Process creation event analysis
* Vulnerability assessment
* Screenshot evidence collection
* Security documentation
* SOC analyst investigation workflow

## Repository Purpose

The purpose of this repository is to demonstrate practical cybersecurity skills through real hands-on lab projects. Each project includes documentation, screenshots, and evidence of completed technical work.
